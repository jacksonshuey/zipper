# Zipper

A universal, LLM-assisted schema-reconciliation engine: point it at any source — a CRM export, a vendor CSV, a JSON API — and for every incoming column it decides whether to **join** an existing canonical column, **append** a new one, or flag it **unclear** for review. It makes that routing decision once per `(record, source, column)`, caches it, normalizes each value to the canonical type, and writes a wide reconciled row plus an append-only audit of why every decision was made. Storage, the LLM router, and the deterministic lookup tier are all pluggable seams, so it drops into any project without a fork.
